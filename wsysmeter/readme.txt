Windows only!
The program is designed to replace the now not working widgets on Windows10. Shows the clock, date, percentage of CPU, RAM, SWAP and hard drives (used on disk / disk capacity / percentage of occupied).
For compile in Lazarus need BGRA controls
http://wiki.freepascal.org/BGRAControls

Только для Windows!
Программа предназначена для замены ныне не работающих widgets на Windows10. Показывает часы, дату, процент занятости CPU, RAM, SWAP и жестких дисков (занято на диске / объем диска / процент занятого).
Для компиляции необходимы компоненты BGRA
http://wiki.freepascal.org/BGRAControls
