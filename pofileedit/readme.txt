Program Localization File Editor
The editor is focused on translating .po files and has a built-in interface for working with Yandex-translator. It works under Windows and Linux.


Редактор файлов локализации программ 
Редактор ориентирован на перевод файлов .po и имеет встроенный интерфейс работы с Яндекс-переводчиком. Работает под Windows и Linux.
